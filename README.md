# Whats the price of Bitcoin!?

Simple Sinatra app that displays the price of bitcoin using the Coin Market Cap API.

### Installing

Install dependencies using bundler

```
bundle
```

Run the server 

```
thin start
```

Navigate to localhost:3000/

## Running the tests

To run the tests

```
rake test
```
